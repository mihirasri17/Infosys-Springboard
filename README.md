Title : HotelRevAI - AI Driven Revenue Analysis for Hotels. The Main Objective is to Understand Customer Satisfaction , identity trends inratings, and highlight key factors influencing hotel performance through interactive dashboards. The project involves data cleaning using Power Query, creating data models, and building insightful visualizations with DAX measures. The dashboard provides features such as overall rating analysis, customer satisfaction insights, location-based performance, and service quality evaluation, enabling users to explore data through filters and slicers. The insights derived help in understanding customer preferences and improving decision-making for better hotel management and enhanced user experience.
